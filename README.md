# Car Rental Project

## Requirements
- Node 12+
- Yarn 1.19+

## Commands
- Install dependencies: ``yarn``
- Start development server: ``yarn dev`` ( port `:8080` )
- Build project: ``yarn build``
( builds to `/public` directory )

## Technologies used

### Front-End
- Bootstrap
- JQuery

### Development
- Node
- Rollup
- Babel
- Postcss
